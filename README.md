# AutoRubric AI

**Intelligent Answer Evaluation System**

AutoRubric AI is an automated, AI-driven evaluation tool designed to provide intuitive, visual, and human-friendly feedback on student assignments. By leveraging a strict deterministic validation layer combined with a Large Language Model (LLM), it moves beyond simple scoring to offer deep insights, structured rubrics, and explainable reasoning for every evaluation.

## Overview

Most AI evaluation tools overwhelm users with unpredictable, unstructured text. AutoRubric AI solves this by strictly enforcing JSON outputs from the LLM and processing them through a robust validation layer. It analyzes student answers against assignment prompts and returns a comprehensive evaluation dashboard that includes:

1.  **Overall Score**: A quick, visual representation of the student's performance, mathematically computed using a weighted system.
2.  **Performance Metrics**: An accurate radar chart plotting the exact scores for Clarity, Depth, Structure, and Relevance.
3.  **Detailed Breakdown**: Animated linear progress bars that instantly highlight strengths and weaknesses.
4.  **Constructive Feedback**: A meaningful, human-readable explanation generated by the AI.
5.  **Structured Rubric**: A transparent, tabular evaluation matrix detailing exactly how the score was calculated.

---

## Architecture

The project follows a decoupled client-server architecture:

-   **Frontend (Vercel)**: A modern, responsive web interface built with HTML5, vanilla JavaScript, Tailwind CSS, and Chart.js.
-   **Backend (Render)**: A REST API built with Python and FastAPI. It integrates with HuggingFace via strict JSON prompting, applies statistical validation, mathematically calculates final scores, and stores results in a SQLite database.
-   **Evaluation Engine**: A hybrid deterministic and AI system. It uses robust statistical metrics (punctuation stripping, stop-word removal) as a fallback layer if the AI is unreachable or returns malformed data.

## Installation and Setup

### Prerequisites

-   Python 3.8+
-   Git

### 1. Clone the Repository
```bash
git clone https://github.com/harsha3358/AutoRubric-AI.git
cd AutoRubric-AI
```

### 2. Set Up the Backend
Navigate to the backend directory and install dependencies:
```bash
cd backend
python -m venv venv
# On Windows: venv\Scripts\activate
# On Mac/Linux: source venv/bin/activate
pip install -r requirements.txt
```

### 3. Configure Environment Variables (Optional)
To enable the AI evaluation, you must set your HuggingFace API key. Create a `.env` file in the `backend` directory:
```env
HF_TOKEN=your_huggingface_token_here
```
*(If omitted, the system will seamlessly fall back to using its deterministic statistical engine).*

### 4. Running the Application
Start the backend server:
```bash
python main.py
# Or use: uvicorn main:app --reload --host 0.0.0.0 --port 10000
```
The backend API will be available at `http://localhost:10000`.

To run the frontend, open a new terminal, navigate to the `frontend` folder, and start a basic server:
```bash
cd frontend
python -m http.server 8000
```
Open your browser to `http://localhost:8000`.

## Evaluation Metrics & Formula

The system evaluates answers across four key dimensions. The final score is calculated mathematically by the backend using this specific weighting:

- **Relevance (40%)**: Alignment with the prompt and keyword overlap (excluding stop words).
- **Depth (25%)**: Level of detail, completeness, and length of the response.
- **Structure (20%)**: Logical flow, sentence formation, and organization.
- **Clarity (15%)**: Readability, vocabulary, and articulation.
