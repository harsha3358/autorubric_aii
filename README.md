## 🧠 Why It Feels Different

<div style="background:#f5f5f5;padding:15px;border-radius:10px">

Most AI tools overwhelm users with raw output.

AutoRubric AI focuses on:

* ✨ Clarity over complexity
* 🎯 Focused feedback
* 🧠 Explainable reasoning
* ⚡ Smooth interaction

</div>
